## CareerGuider
CareerGuider is a mockup for a careerguider application which helps to choose streams and courses after 10th and 12th respectively.

### System Requirements

1. Android Studio (Above 3.0).
2. API Level >23.


### Steps to run
1. Download/Clone the repository to your local system.
2. Just open existing project with your Android Studio.
